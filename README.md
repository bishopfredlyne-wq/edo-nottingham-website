# Edo State Diaspora Nottingham Website

A premium React/Vite website for Edo State Diaspora Nottingham Chapter.

## Features
- Professional homepage
- About section
- Culture and values section
- Programmes section
- Gallery placeholders
- Leadership section
- Membership call-to-action
- Contact form design
- Mobile responsive layout

## How to run locally

1. Install Node.js from https://nodejs.org
2. Open this folder in VS Code or Terminal
3. Run:

```bash
npm install
npm run dev
```

## How to deploy publicly on Vercel

1. Create a GitHub account
2. Create a new repository
3. Upload all these files
4. Go to https://vercel.com
5. Import the GitHub repository
6. Click Deploy

Your site will become public with a Vercel link.
